Recent Projects

Tunnel remote monitoring - Switzerland

At ParaV we designed geotechnical monitoring system with the help of Java, Postgre SQL and DeviceHive to detect any incident that may affect the tunnel works.

IoT Solutions include:

  • The remote monitoring system covers 55 load anchors and five extensometers, all controlled from a central gateway outside the tunnel.
  • The system's wireless operation and low power consumption offered a minimum-impact way of guaranteeing that the system does not interfere with construction work.

ePill - Boston Farm - Real time health monitoring for livestock health

Utilising Apache Kafta, Postgre SQL with Agile methodology, ParaV helped save costs for cattle farmers with instant real time monitoring of diseased cattle's.

IoT Solutions include:

  • ePill that is implanted into the cattle
  • State machine code, Acquiring sensor data using I2C, analog and digital IO
  • Basic status displays based on link availability, heartbeat mechanisms

London City - Flood Management system

Using Agile along with React Native, DeviceHive and Java ParaV was able to create a localised early-warning flood system.

IoT Solutions include:

  • Ultrasonic Sensors in the stream groundwater and basin of the Thames and Cherwell rivers.
  • Geo visualisation and data analytic tool.
  • The use of TV white space to connect a number of hard-to-reach devices in the network.
